# 🎙️ Edge TTS Studio

A beautiful, free, and lightweight web application to convert text to natural-sounding speech using Microsoft Edge's neural voices.

![UI Preview](https://github.com/anhnhatdev/edge-tts-studio/raw/main/preview.png)

## ✨ Features
- **Neural Voices**: Access 40+ high-quality English voices from Microsoft.
- **100% Free**: No API keys required, no character limits.
- **Customizable**: Adjust speed, pitch, and volume.
- **Modern UI**: Glassmorphism dark mode design.
- **History**: Keeps track of your recent generations.
- **Downloadable**: Save your speech as high-quality MP3 files.

## 🚀 Quick Start

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the App
```bash
python app.py
```
Open [http://localhost:8000](http://localhost:8000) in your browser.

## 🛠️ Tech Stack
- **Backend**: FastAPI (Python)
- **TTS Engine**: `edge-tts`
- **Frontend**: HTML5, Vanilla CSS, Vanilla JS
- **UI Style**: Glassmorphism / Dark Mode
